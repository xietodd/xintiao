# 💰 薪金矿工 - 带薪摸鱼财富可视化

## 🎯 项目简介
"薪金矿工"是一款专为打工人设计的趣味网页应用，将您的工作时间实时转化为看得见的金钱增长，让您直观感受"时间就是金钱"的真理！

## 🚀 使用方法

### 1. 快速启动
- 双击打开 `salary_miner.html` 文件
- 首次使用需输入您的薪资信息
- 支持日薪和时薪两种计算模式

### 2. 功能特色
- **实时计算**：每秒更新您的累计收入
- **金币动画**：金币和钞票从屏幕飘落，点击金币有惊喜音效
- **老板键**：按 `Esc` 键瞬间切换到代码编辑器伪装界面
- **本地存储**：所有数据保存在浏览器本地，隐私安全
- **响应式设计**：支持手机、平板、电脑各种屏幕

### 3. 操作指南

| 功能 | 操作方式 |
|------|----------|
| 开始赚钱 | 输入薪资后点击"开始赚钱" |
| 暂停/继续 | 右上角控制面板 |
| 静音/开音 | 右上角控制面板 |
| 老板键 | 按键盘 `Esc` 键 |
| 重置设置 | 右上角控制面板 |
| 查看帮助 | 右上角控制面板 |

### 4. 安全提示
- ✅ 所有数据仅存储在本地浏览器
- ✅ 不会上传到任何服务器
- ✅ 老板键伪装成代码编辑器界面
- ⚠️ 请合理使用，注意职场安全

## 🎮 趣味玩法

1. **金币收集**：点击掉落的金币获得额外音效
2. **时间感知**：观察每分钟实际收入
3. **成就挑战**：看多久能赚回一杯咖啡钱
4. **摸鱼计时**：精确记录每次摸鱼时长

## ⚡ 技术特性
- 纯HTML单文件，无需安装
- 使用CSS3动画，流畅不卡顿
- Web Audio API生成真实音效
- localStorage本地数据持久化
