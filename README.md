# 💰 薪金矿工 - 带薪摸鱼财富可视化

## 🎯 项目简介
"薪金矿工"是一款专为打工人设计的趣味网页应用，将您的工作时间实时转化为看得见的金钱增长，让您直观感受"时间就是金钱"的真理！

## 🚀 使用方法

### 1. 快速启动
- 双击打开 `salary_miner.html` 文件
- 首次使用需输入您的薪资信息
- 支持日薪和时薪两种计算模式

### 2. 功能特色
- **实时计算**：每秒更新您的累计收入
- **金币动画**：金币和钞票从屏幕飘落，点击金币有惊喜音效
- **老板键**：按 `Esc` 键瞬间切换到代码编辑器伪装界面
- **本地存储**：所有数据保存在浏览器本地，隐私安全
- **响应式设计**：支持手机、平板、电脑各种屏幕
- **⚙️ 设置面板**：完整的参数调节系统，支持所有功能自定义

### 3. 操作指南

| 功能 | 操作方式 |
|------|----------|
| 开始赚钱 | 输入薪资后点击"开始赚钱" |
| 暂停/继续 | 右上角控制面板 |
| 静音/开音 | 右上角控制面板 |
| 老板键 | 按键盘 `Esc` 键 |
| 重置设置 | 右上角控制面板 |
| 查看帮助 | 右上角控制面板 |
| ⚙️ 设置面板 | 点击右上角⚙️图标 |

### 4. ⚙️ 设置面板功能

**💰 薪资设置**
- 薪资模式切换（时薪/日薪）
- 薪资金额输入

**⏰ 工作时间设置**
- 上班时间设置
- 下班时间设置
- 实时计算工作时长

**🎭 动画效果设置**
- 金币掉落频率（1-10个/秒）
- 钞票掉落频率（0-5个/秒）
- 动画速度调节（0.1x-3x）

**📊 显示设置**
- 更新频率（50-1000毫秒）
- 小数位数（0-4位）

**🔊 音效设置**
- 音量调节（0-100%）
- 音调调节（0.5x-2x）

### 5. 安全提示
- ✅ 所有数据仅存储在本地浏览器
- ✅ 不会上传到任何服务器
- ✅ 老板键伪装成代码编辑器界面
- ⚠️ 请合理使用，注意职场安全

## 🎮 趣味玩法

## ⚡ 技术特性
- 纯HTML单文件，无需安装
- 使用CSS3动画，流畅不卡顿
- Web Audio API生成真实音效
- localStorage本地数据持久化
- 完整的设置面板系统，支持实时参数调节
- 响应式布局，适配各种屏幕尺寸
